# Pokédex

Pokédex es una aplicación que nos muestra la imagen y detalle de cada pokémon además de hacer una búsqueda por el nombre.
Para realizar esta web se utilizó:

* jquery-3
* components
* bootstrap3
* javascript
* API de pokeapi
